
Usage: 
    todo [command] [required-argument] [option] [argument]

Without arguments the command defaults to listing open tasks.

Commands:
    
    help | h                                    Display this help.

    version | v                                 Display program version.

    count                                       Display task count. 
                                                A switch is optional.

        --all           | -a
        --completed     | -c
        --due           | -d
        --inprogress    | -ip
        --onhold        | -h
        --open          | -o                        default
        --overdue       | -od 

    list | l                                    List tasks or groups. 
                                                A switch is optional.

        --all           | -a
        --completed     | -c
        --deleted       
        --due           | -d
        --inprogress    | -ip
        --onhold        | -h
        --open          | -o                        default
        --overdue       | -od 
        
    add | a <summary>                           Add a new task. 
                                                Summary is required (max 255 characters).
                                                Additional arguments are optional.

        --description           <text>
        --due [date]    | -d    <date>              yyyymmdd or yyyy-mm-dd
        --group         | -g    <text>              max 24 characters
        --parent                <task_id>
        --priority      | -p    <number>            default 500

    show | s <task_id>                          Display task details.

    update | u <task_id>                        Update task. 
                                                At least one argument is required.

        --description           <text>
        --due           | -d    <date>              yyyymmdd or yyyy-mm-dd
        --group         | -g    <text>              max 24 characters
        --parent                <task_id>
        --priority      | -p    <number>            default 500
        --summary       | -s    <text>              max 255 characters
                                                
    delete | del <task_id>                      Delete task.

    complete | c <task_id>                      Set task completed.
                                                Optional closing comment can be added.

        --comment       | -d    <text>              

    reopen | open <task_id>                     Reopen completed task.

    configure                                   Add default config and database, if they don't exist.
        --local                                     do it in current directory (for testing)
        --reset                                     remove existing config and database, if they exist


Examples:
    todo
    todo a "New task"
    todo add "New task" --due=2024-08-13 -pty=3
    todo list --all
    todo l -o
    todo count -c
    todo update 15 --priority=1 --group="Project"
    todo u 10 --due=
    todo c 12
    todo reopen 3
    todo del 5
